# Suggested Commands

## Testing
- Run all tests: `pnpm test`
- Run specific test file: `pnpm test -- tests/engine/vec2.test.ts`
- Run tests matching pattern: `pnpm test -- tests/ai/`

## Type Checking
- `pnpm exec tsc --noEmit`

## Development
- `pnpm dev` - Start Vite dev server
- `pnpm build` - Production build

## System Commands (Windows with bash shell)
- Use forward slashes in paths
- Use /dev/null not NUL
- Standard Unix shell syntax (ls, grep, find)
