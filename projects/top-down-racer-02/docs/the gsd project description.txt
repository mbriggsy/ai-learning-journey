A top-down racing game with an AI training pipeline.
Game: TypeScript + PixiJS. Two-layer architecture — a pure-logic Simulation Engine (physics, collision, track geometry, car dynamics) completely decoupled from a PixiJS Renderer (visual layer only). The engine runs headless in Node.js for AI training at thousands of sims/sec. The renderer never touches game logic.
Physics: Simcade. Analog inputs — Throttle (0–100%), Brake (0–100%), Steering (-1.0 to +1.0) with keyboard smoothing. No dedicated drift system — natural oversteer from physics only. Fixed 60Hz tick, decoupled from rendering. Custom deterministic physics (no external physics engine — determinism required for AI training).
Tracks: Spline-based centerline with width. Closed loops only. 1 track for early phases, 3–5 later. Checkpoint gates along spline (20–50 per track, crossed in order). Three-layer surfaces: road → soft runoff (reduced grip) → hard wall. Slide along walls with speed penalty proportional to impact angle. Ghost cars — no car-to-car collision.
AI Pipeline: Gymnasium-compatible wrapper via Node.js ↔ Python bridge (ZeroMQ/WebSocket). 9 rays across 180° forward arc (22.5° intervals). 14-value observation vector: 9 ray distances + speed + angular velocity + steering angle + normalized lap progress + distance from centerline. Dense rewards every tick — checkpoint progress (primary) + speed bonus, no centerline bonus. Four-tier penalties: stillness timeout, wall contact, off-track, backward driving. Penalties always smaller than progress rewards. ML via stable-baselines3 + PyTorch (PPO/SAC).
Game Features: Clean HUD (lap timer, best lap, lap counter, speedometer, minimap). No damage/health. Medium polish — skid marks, dust/spark particles, engine sound with pitch-shift. No difficulty settings.
AI vs Human Mode: Simultaneous racing + ghost replay. Spectator/demo mode (watch AI solo). Local leaderboard only — best lap per track for human and AI.
Six phases: (1) Core simulation engine, (2) PixiJS renderer + playable game, (3) Game features & polish, (4) Gymnasium environment wrapper, (5) AI training pipeline, (6) AI vs Human mode.
